# MindBridge User Flows

## Flow 1: Book a Counselling Session

Home Page → Book Appointment → Select Counsellor → Select Date → Select Time Slot → Add Optional Reason → Confirm Booking

This flow supports students who need quick access to counselling without going through a complex administrative process.

## Flow 2: Track Daily Mood

Home Page → Mood Tracker → Select Emoji Mood → Select Mood Influence Tags → Add Optional Journal Entry → Save Entry → View Weekly Mood Trend

This flow supports quick emotional check-ins while still allowing deeper reflection through the optional journal field.

## Flow 3: Access Mental Health Resources

Home Page → Resources → Select Category → Review Resource Card → Open Relevant Resource

This flow helps students find topic-based support material without browsing through a long undifferentiated list.

## Flow 4: Login or Register

Home Page → Login/Register → Choose Login or Register Tab → Enter University Email → Enter Password → Submit

This flow supports secure student access while keeping the authentication process familiar and simple.
