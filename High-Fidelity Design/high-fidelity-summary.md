# High-Fidelity Mockups
The high-fidelity screens were created in Figma after the low-fidelity 
wireframes were reviewed. These screens apply the final colour palette, 
typography, layout system and component styling.
## Screens Included- Home / Landing Page- Login and Register Page- Appointment Booking Page- Mood Tracker Page- Resources Page
## Purpose
The high-fidelity prototype demonstrates the final visual direction and interaction flow of MindBridge. It also supports evaluation using usability 
heuristics.
## Key Improvements from Low-Fidelity Stage- Final colour palette applied.- Typography system added.- Buttons and cards styled consistently.- Appointment booking form refined.- Mood tracker visual feedback added.- Resource categories visually organised
