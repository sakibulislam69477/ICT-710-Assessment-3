# Low-Fidelity Wireframes

The low-fidelity wireframes were created to explore the basic layout, information hierarchy and user flow before moving into high-fidelity Figma design.

## Pages Included

- Home screen
- Login and registration screen
- Book a session screen
- Mood tracker screen
- Resources screen

## Purpose

The purpose of this stage was to confirm the structure of each MVP page and ensure that the main student tasks were easy to understand before visual styling was applied.

## Design Focus

The low-fidelity stage focused on:

- Basic page layout
- Navigation structure
- Placement of call-to-action buttons
- Form structure for booking and login
- Mood tracker flow
- Resource category organisation
