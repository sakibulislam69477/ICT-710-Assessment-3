# MindBridge: University Student Mental Health & Wellness Portal

MindBridge is a web-based mental health and wellness platform designed for university students. The prototype supports five core MVP pages:

- Home / Landing Page
- Login and Register Page
- Appointment Booking Page
- Mood Tracker Page
- Resources Page

This repository stores the design evidence, wireframes, high-fidelity mockups, usability evaluation notes, revision evidence.


## Project Goal
The goal of MindBridge is to provide a calm, private and accessible digital 
space where university students can access mental health support, book 
counselling sessions, track their mood and explore curated wellbeing resources.
## Target Users
## MVP Features
The primary users are university students aged 18-25 who may prefer digital 
self-service support before accessing in-person counselling services.
1. Landing page with clear value proposition and call-to-action buttons.
2. Login and registration page using university credentials.
3. Appointment booking page with counsellor selection, date and time slots.
4. Mood tracker with emoji-based check-in and weekly mood trend.
5. Resource library organised by mental health topics.
